# Glitch - I/O tool!
A work-in-progress I/O script for EA Canada(now EA Vancouver)'s not-so-alphanumeric ReBoot game PlayStation 1 game using reverse engineering. Unless a miracle happens, ReBoot: Countdown to Chaos happens to be [one of] the only source(s) for "legit" ReBoot assets. This is why the recovery of such assets has been a goal.


## Features
Currently, keytool only allows for input of .TOM (Texel Object Map) models from the PS1 game(and only ReBoot video game to exist) ReBoot: Countdown to Chaos.
